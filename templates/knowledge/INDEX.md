## Claude 행동 규칙

### Step 1: 키워드 검색
```
사용자 요청에서 주제 키워드 추출
→ 위 "키워드 → MOC 매핑" 테이블에서 검색
```

### Step 2: 결과에 따른 행동
```
매칭됨:
  → 해당 MOC 읽기
  → "기존에 [주제] 관련 자료가 있습니다: 논문 N개"

매칭 안 됨 (또는 테이블에 "_없음_"만 있음):
  → "새로운 주제입니다. 조사를 시작할까요?"
```

### Step 3: 작업 후 업데이트
```
새 지식 저장 시:
  1. "최근 업데이트"에 행 추가 (10개 초과 시 오래된 것 삭제)
  2. "통계" 숫자 증가

새 MOC 생성 시:
  1. "키워드 → MOC 매핑"에 행 추가 (동의어 모두 포함)
  2. "_없음_" 행이 있으면 삭제

새 프로젝트 연결 시:
  1. "프로젝트 → 주제 매핑"에 행 추가
  2. "_없음_" 행이 있으면 삭제
```

### MOC 생성 기준
- 해당 주제 논문 **2개 이상** 저장 시
- 새 프로젝트가 해당 주제를 다룰 때
- 사용자가 명시적으로 요청 시

### 동의어 찾기 (MOC 생성 시)
```
새 MOC 생성 시 키워드 등록 절차:
1. 주제의 정식 명칭 (예: depth from defocus)
2. 약어 (예: DfD, DoF)
3. 관련 용어 (예: blur, bokeh, circle of confusion)
4. 한글 용어 (있다면)
→ 모두 하나의 MOC에 매핑
```
