# Deprecated Compatibility Asset

This legacy top-level review packet template is no longer used for new projects.

Use:

- `templates/shared/reviews/README.md`
- `templates/shared/reviews/cycles/REVIEW_PACKET.md`
- `templates/shared/reviews/cycles/GPT_REVIEW.md`
- `templates/shared/reviews/cycles/NEXT_PROMPT.md`

In v2, scaffolded repos create cycle documents under `reviews/cycles/CYCLE-####/` and use repo-local skills instead of treating review helpers as always-loaded instructions.
