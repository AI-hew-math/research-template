# Knowledge Base

Shared knowledge assets for template-based repositories.

## Contents

- `papers/` - paper notes
- `lessons_learned.md` - cross-project lessons

## Naming

- Paper notes should use `{Author}_{Year}_{Keyword}.md`.

## Expansion Rules

- Create a topic MOC when 10 or more papers accumulate around one topic.
- Create a shared `methods/` area when 3 or more projects reuse the same method.
