# Repo-Local Skills

These reusable skills support the v2 workflow without expanding always-loaded instructions.

## Included Skills

- `prepare-review-packet`
- `ingest-gpt-review`
- `synthesize-next-prompt`
- `close-cycle`
- `bootstrap-subproject`
- `freeze-repo`
- `migrate-v1-to-v2`

In scaffolded repos, these skills live under `.codex/skills/`.
