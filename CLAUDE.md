# Research Template - Phase 0: 탐색 모드

> 이 폴더에서 agent를 실행하면 **연구 탐색 모드**입니다.
> 프로젝트를 만들기 전, 아이디어를 구체화하는 단계입니다.

---

## Agent 행동 지침

### 세션 시작 시

사용자에게 다음 질문들로 연구 방향을 파악합니다:

```
1. "어떤 분야/주제에 관심이 있으세요?"
2. "해결하고 싶은 구체적인 문제가 있나요?"
3. "관련해서 읽은 논문이나 떠오르는 아이디어가 있나요?"
4. "이 연구로 달성하고 싶은 목표가 뭔가요?"
```

**한 번에 다 묻지 말고**, 대화하면서 자연스럽게 파악합니다.

---

### 탐색 중 할 일

| 사용자 요청 | Agent 행동 |
|------------|------------|
| 논문 조사 요청 | WebSearch로 조사 → `../_knowledge/papers/`에 저장 |
| 아이디어 논의 | 장단점 분석, 관련 연구 연결 |
| 방향 고민 | 옵션 제시, 트레이드오프 설명 |

---

### 프로젝트 생성 타이밍

다음 조건이 갖춰지면 프로젝트 생성을 제안합니다:

1. **연구 주제**가 명확해짐
2. **핵심 아이디어/가설**이 정리됨
3. **실험 방향**이 대략 잡힘

```
Agent: "방향이 잡힌 것 같아요. 정리하면:
  - 주제: [주제]
  - 핵심 아이디어: [아이디어]
  - 예상 실험: [실험]

  프로젝트 만들까요?"

사용자: "응"

Agent: "프로젝트 이름 뭘로 할까요? 제안: [제안명]"

→ ./create_project.sh "[이름]" "[설명]" 실행
→ "cd ../[이름] && claude 하시면 됩니다" (사용 중인 agent CLI)
```

---

### 아직 방향이 안 잡혔을 때

- 급하게 프로젝트 만들지 않아도 됨
- 탐색 내용은 `../_knowledge/papers/`에 저장되어 보존됨
- 다음 세션에서 이어서 탐색 가능

---

## 유용한 명령어

```bash
# 프로젝트 생성
./create_project.sh "ProjectName" "설명"

# 지식 베이스 확인
ls ../_knowledge/papers/
```
